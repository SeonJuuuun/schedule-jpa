### 개발 전, 공통 조건 

- [ ] 모든 테이블은 고유 식별자(ID)를 가집니다.
- [ ] `3 Layer Architecture` 에 따라 각 Layer의 목적에 맞게 개발합니다.
- [ ] CRUD 필수 기능은 모두 데이터베이스 연결 및  `JPA`를 사용해서 개발합니다.
- [ ] `JDBC`와 `Spring Security`는 사용하지 않습니다.
- [ ] 인증/인가 절차는 `JWT`를 활용하여 개발합니다.
- [ ] JPA의 연관관계는 `양방향`으로 구현합니다.
